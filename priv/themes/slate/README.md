## Installation

1. Open Zed.
2. Press `ctrl` + `shift` + `p` and select *zed: extensions*.
3. Select *Slate* and press *Install*.
4. Press (`ctrl` + `k`, `ctrl` + `t`) to open the theme selector and select *Slate Dark* or *Slate Light*.
