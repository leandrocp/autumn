## Installation

1. Open `Command Palette`
2. Select `zed: extensions`
3. Search `Colorizer`

## Activate Theme

1. Open `Command Palette`
2. Select `theme selector: toggle`
3. Search `Colorizer` or `Colorizer Pro`
