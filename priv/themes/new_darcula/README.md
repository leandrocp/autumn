### Install via Zed Extensions
1. Open Zed
2. `cmd+shift+p` and select *zed: extensions*
3. Search/select *New Darcula Theme* and Install

### Install Manually
1. Download [newdarcula.json](./themes/newdarcula.json)
2. Put into `~/.config/zed/themes/`
