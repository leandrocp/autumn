# Lusch
`variants: [Lusch, Darker]`
A dark colorful theme for zed based off the colors used in the Discord UI

## Installation
1. Copy lusch.json to ~/.config/zed/themes/
2. Open Zed and navigate to Settings -> Theme -> Lusch
