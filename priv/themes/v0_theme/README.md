# v0 theme
`variants: [Dark, Light]`

A port of the theme used in [v0 chat](https://v0.dev/chat/) for zed.

## Installation
1. Copy v0.json to ~/.config/zed/themes/
2. Open Zed and navigate to Settings -> Theme -> V0
