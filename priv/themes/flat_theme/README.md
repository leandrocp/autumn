# Flat
Minimal, easy on the eyes theme inspired by [flatwhite syntax](https://github.com/biletskyy/flatwhite-syntax) from Atom

## Installation
1. Copy flat.json to ~/.config/zed/themes/
2. Open Zed and navigate to Settings -> Theme -> flat
